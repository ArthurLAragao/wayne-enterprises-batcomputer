# Wayne Enterprises - Batcomputer Grid

Advanced Monitoring Grid para Wayne Enterprises - Um painel de controle tatico inovador inspirado no universo do Batman.

---

## Sobre o Projeto

O Batcomputer Grid eh uma aplicacao web tematica que simula um painel de monitoramento avancado das industrias Wayne. Inspirado na interface do Batman, combina:

- Design Noir-Corporativo com ouro e ciano (Wayne Enterprises)
- - Interface com monitoramento de areas, inventario e pessoal
  - - Personagens Batman integrados ao sistema
    - - Login com controle de acesso e auditoria
     
    - ## Como Abrir o Arquivo
   
    - ### Opcao 1: Direto no Navegador
   
    - 1. Faca download do arquivo batman-projeto-2.0.html
      2. 2. Duplo-clique no arquivo para abri-lo
         3. 3. Pronto! A interface estara carregada
           
            4. ### Opcao 2: Servidor Local (Recomendado)
           
            5. python -m http.server 5500
            6. # ou
            7. npx live-server
           
            8. Depois acesse http://localhost:5500 no navegador
